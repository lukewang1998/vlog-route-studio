# Vlog Route Studio

A reusable travel vlog route framework. Each destination lives in its own folder and can include bundled photos, maps, shooting checklists, and per-location script notes saved in the browser with `localStorage`.

## Destinations

- `dubai/`: Dubai 4-day vlog shooting route.

## GitHub Pages

Publish from the repository root. The entry file is `index.html`, so GitHub Pages can serve the destination hub directly from the `main` branch.
